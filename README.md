# CEP-Finder

Um buscador de CEP rápido e fácil com integração de API.

## 📋 Sobre o projeto

O **CEP-Finder** é uma aplicação web que permite pesquisar endereços a partir de um CEP (Código de Endereçamento Postal) brasileiro. Basta digitar o CEP desejado e a aplicação retorna automaticamente as informações do endereço correspondente, como rua, bairro, cidade e estado, utilizando integração com APIs de consulta de CEP.

## ✨ Funcionalidades

- 🔍 Busca de endereço por CEP
- ⚡ Resposta rápida via integração com API
- 🗺️ Exibe rua, bairro, cidade e estado
- 💻 Interface simples e intuitiva

## 🚀 Como usar

1. Acesse o site (link acima quando disponível).
2. Digite um CEP válido no campo de busca (ex: `01310-100`).
3. Clique em **Buscar** e visualize o endereço completo.
